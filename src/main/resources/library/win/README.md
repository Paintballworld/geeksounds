# Win Jingles

Place celebratory sound effects here for when players guess correctly!

## Supported Formats
- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)
- M4A (.m4a)

## Examples
- Crowd cheering
- Victory fanfare
- Applause
- "Correct!" sound effects
- Game show success sounds
- Ta-da sounds

## Tips
- Keep jingles short (2-5 seconds)
- The game will randomly select one from this folder each time someone guesses correctly
- Having 3-5 different win sounds adds variety and keeps the game exciting
- Make sure volume levels are consistent across all files
