# Sound Files

Place your sound trivia files here!

## Supported Formats
- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)
- M4A (.m4a)

## Naming Convention
Use descriptive names with underscores or hyphens. The file name will be displayed to players after they guess.

Examples:
- `windows_95_startup.mp3` → "Windows 95 Startup"
- `mario-coin.wav` → "Mario Coin"
- `AOL_you_got_mail.mp3` → "Aol You Got Mail"

## Tips
- Keep sounds between 3-10 seconds for best gameplay
- Mix difficulty levels (easy iconic sounds and harder obscure ones)
- Organize by themes if you want (80s games, movie quotes, etc.)
